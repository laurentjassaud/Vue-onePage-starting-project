# Vue-onePage-starting-project
Mise en place rapide pour mini site avec composants vue.js.
Le fichier config.json permet de definir les élément spécifiques.

## based from 
[Designstub](http://www.designstub.com/)

### Utilise :
* vue-cli
* webfontloader
* gsap

### avec
* bootstrap 4 scss
* fontAwesome scss
